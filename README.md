# 授業登録・担当表一元管理システム (Schedule-App)

## 概要
個別指導塾における講師の外部システムへの授業登録状況と、担当生徒・スケジュールをローカル環境で一元管理するためのデスクトップアプリケーション。
運営体制の変更に伴う教室長の管理負担を軽減するため、「誰が・いつ・誰を担当しているか」「振替等で対応が必要なタスクは何か」を一目で直感的に把握できるUIを提供。

## 開発の背景と目的
* **入力漏れの防止:** 月末までに完了すべき授業実績のシステム入力進捗を可視化し、入力漏れを防ぐ。
* **管理コストの削減:** 新しい教室長が着任した際でも、紙や複数システムに散らばりがちな担当情報を1つの画面で俯瞰できるようにする。
* **イレギュラーへの対応:** 振替授業が未定のまま放置されることを防ぎ、代行講師が担当する場合でも元の担当講師にデータを紐づけて情報を整理する。

## 主要機能
* **ダッシュボード:** 全講師の月間入力進捗と、振替未定などの「要対応アラート」を一覧表示。
* **時間割表:** 曜日×時限のスケジュール表。任意のコマをクリックすると、右側の詳細パネルに該当時間の全授業データが展開されます。
* **検索・一覧機能:** 講師名や生徒名から、現在の担当授業を瞬時に検索。
* **マスタ管理機能:** 講師、生徒（学年含む）、および授業枠（月間の目標コマ数）のCRUD（登録・変更・削除）操作。
* **自動バックアップ:** アプリケーション終了時にJSONデータのバックアップを自動生成。

## 使用技術
* **言語:** Python 3.x
* **GUIライブラリ:** PyQt6(予定) （複雑な表レイアウトとモダンなUIの実現）
* **データ保存:** ローカルJSONファイル （外部DB・サーバー不要でPC単体で動作）
